# Contributor Covenant Code of Conduct

## 1. Our Pledge

We pledge to create an open and welcoming environment.

## 2. Our Standards

Examples of positive behavior:

- Respectful communication  
- Graceful acceptance of feedback  
- Focus on what is best for community  

Examples of unacceptable behavior:

- Harassment  
- Trolling  
- Personal attacks  
- Publishing private information  

## 3. Enforcement Responsibilities

Project maintainers enforce rules.

## 4. Scope

Applies to all project spaces and public communication.

## 5. Enforcement

Report issues to:  
conduct@navis.dev
